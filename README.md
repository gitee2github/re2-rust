# re2-rust

a compatible RE2 API(
2021-11-01)  by calling Rust library [regex(rure)](https://github.com/rust-lang/regex)


``` Shell
$ git clone https://gitee.com/openeuler/re2-rust.git
$ cd re2-rust
```

### 编译、安装re2-rust

``` Shell
$ make
$ sudo make install
```
使用rure库:

``` Shell
$ cd regex-capi/ctest/
$ gcc test.c -o test -lrure
$ ./test
```
